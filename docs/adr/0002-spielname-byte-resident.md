# 0002 – Spielname „Byte Resident“

- **Status:** Akzeptiert
- **Datum:** 2026-06-14
- **Entscheider:** Projekt-Team

## Kontext

Das Projekt ist ein Spiritual Successor einer klassischen 80er-Lebenssimulation (*Little Computer People*, Activision 1985). Der ursprüngliche Repository-Name „LittleComputerPoeple“ birgt Marken- und Verwechslungsrisiken. Es wurde ein eigenständiger Spielname benötigt.

Alternativen wurden geprüft:

| Name | Ergebnis |
|------|----------|
| Little Computer People (Remake) | Verworfen – Markenrecht Microsoft/Activision |
| Tiny Life | Verworfen – identisches Life-Sim-Spiel auf Steam (seit 2023) |
| Screenbound | Verworfen – identisches Spiel erscheint Sept. 2026 + Medienmarke UK |
| **Byte Resident** | Akzeptiert – keine Namenskollision gefunden |

## Entscheidung

**Spielname:** Byte Resident  
**Untertitel / Tagline:** Dein kleiner Untermieter im PC

### Positionierung

- **Spiritual Successor**, kein Remake
- Marketing: „inspiriert von klassischen 80er-Computersimulationen“
- Nicht: „Remake von Little Computer People“ oder Nutzung fremder Marken

### Repository

Der Projektordner und Repository-Name lautet **`ByteResident`**. Produktname in Docs und Marketing ist **Byte Resident**.

## Konsequenzen

### Positiv

- Eigenständige Marke, passend zum Konzept (Bewohner im Computer)
- Untertitel erklärt das Spielprinzip auf Deutsch
- Kein direkter Konflikt mit bekannten Spieltiteln (Stand Recherche Juni 2026)

### Negativ / Offen

- Wort „Resident“ kann in Store-Suchen an andere Titel erinnern – kein Horror-Branding verwenden
- Marke noch nicht formal registriert – EUIPO/DPMA/USPTO-Recherche vor Release erforderlich
- Domains (`byteresident.com` etc.) sollten zeitnah reserviert werden

## Marketing-Leitlinien

- Vollständiger Titel: **Byte Resident – Dein kleiner Untermieter im PC**
- Kurzform: **Byte Resident**
- Hashtag-Vorschlag: `#ByteResident`
