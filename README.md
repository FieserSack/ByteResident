# Byte Resident

**Dein kleiner Untermieter im PC**

Digitale Lebenssimulation als Spiritual Successor klassischer 80er-Computersimulationen: Ein autonomer Bewohner lebt in einer Wohnung auf deinem PC – du kümmerst dich um ihn, beobachtest ihn und greifst behutsam ein.

**Status:** Planungsphase
