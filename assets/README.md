# Assets

Grafiken, Audio, Sprites und andere Medien für Byte Resident.

## Struktur (Vorschlag)

```text
assets/
├── sprites/       # Charaktere, Objekte
├── ui/            # Icons, Buttons, HUD
├── audio/         # Musik, Soundeffekte
└── fonts/         # Schriftarten
```

## Richtlinien

- Dateinamen: `kebab-case` (z. B. `character-walk.png`)
- Lizenzen und Quellen dokumentieren
- Große Binärdateien: Git LFS prüfen, sobald Assets anfallen
