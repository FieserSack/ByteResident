# Quellcode

Platzhalter für den Anwendungscode. Die Struktur wird nach der Tech-Stack-Entscheidung (ADR) angelegt.

## Geplante Module

| Modul | Beschreibung |
|-------|--------------|
| `core/` | Spiel-Loop, Tick, globaler Zustand |
| `world/` | Räume, Objekte, Interaktionen |
| `agents/` | Charakter-Logik, KI, Bedürfnisse |
| `ui/` | Darstellung und Spieler-Eingabe |
| `persistence/` | Speichern und Laden |

Siehe [docs/architecture/overview.md](../docs/architecture/overview.md) für Details.
